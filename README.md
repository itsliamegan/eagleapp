# eagleApp

eagleApp was a schedule app I built for my high school in the 2019-20 school
year. Its goal was to show the schedule for the current day, as well as be able 
to look forwards or backwards at upcoming or previous schedules. Unfortunately,
due to a lack of both initiative and messaging on my part, it never saw
widespread adoption outside of a few of my close friends. But, I think helping
some people is better than none, so I'm satisfied with it.

This site home-grown HTTP framework called "Janus", rather than an existing
solution like Symfony or Laravel. This framework was a hodge-podge of my
personal preferences at the time of development, and I quickly realized that
creating a framework that represented my personal tastes at one point in time
was a pretty big mistake. Unfortunately, I deleted the files for this framework
at some point, so the files within this site are no longer functional and
instead only are available for posterity. There are numerous bugs that I found
during the usage of this system, and because I only had FTP access to these
files, I had to fix them by directly editing the files on the production server
and didn't get a chance to backport to my personal copy of the files. Again,
these files are only here for posterity.
